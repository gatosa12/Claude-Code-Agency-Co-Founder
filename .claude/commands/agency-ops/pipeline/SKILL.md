## Pipeline

**Status:** Coming in Phase 2

This skill will help view and manage your deal pipeline by reading:
- `context/agency.md` for agency context
- `context/pipeline/` for pipeline data
- This skill's `learnings.md` for past improvements

### When Ready, This Skill Will:
1. Show deal stages from new lead through closed
2. Move leads between stages and log transitions
3. Calculate conversion metrics (messages-to-replies, calls-to-close)
4. Learn from your feedback to improve future output

Run `/agency-ops:onboarding` first to set up your agency context.
